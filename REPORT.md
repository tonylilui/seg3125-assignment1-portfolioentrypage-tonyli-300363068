### c. Screen Layout

**Layout Description:**

My portfolio uses a **single-page, vertically-scrolling layout** with three distinct sections: About Me, How I Work, and Case Studies. The layout incorporates the following principles from NN/g visual communication theory:

#### Scale

- **Large hero heading** ("Hi, I'm Tony Li") uses a display-3 class with responsive font sizing (`calc(2rem + 3vw)`) to immediately capture attention
- **Section headings** use display-5 class to establish clear section boundaries
- **Card titles** are smaller but still bold, creating a clear size hierarchy
- The progressive scaling guides the eye from most to least important information

#### Visual Hierarchy

- **F-pattern reading flow:** The layout follows the natural F-pattern of web reading, with the most important information (name, tagline) at the top-left
- **Color contrast:** Primary blue buttons and links draw attention to interactive elements
- **Whitespace:** Generous padding and margins create breathing room and separate content groups
- **Typography weight:** Bold headings, regular body text, and muted secondary text create clear information levels
- **Section backgrounds:** Alternating light/dark backgrounds visually separate the About and Case Study sections

#### Balance

- **Symmetrical balance:** The hero section uses a 50/50 split between text content and profile image placeholder
- **Grid-based layout:** The case studies section uses Bootstrap's 12-column grid system with equal-width cards for visual equilibrium
- **Centered content:** The "How I Work" section centers all content horizontally, creating a focal point
- **Footer balance:** The footer is centered and minimal, providing a clean visual anchor at the bottom

#### Contrast

- **Color contrast:** Dark text on light backgrounds ensures WCAG AA compliance for readability
- **Typography contrast:** Serif headings against sans-serif body text create visual interest and distinction
- **Shape contrast:** Rounded cards and circular profile image contrast with the rectangular layout structure
- **Interactive contrast:** Buttons have hover effects (lift and shadow) that provide clear visual feedback
- **Section contrast:** The alternating background colors between sections help users understand content boundaries
